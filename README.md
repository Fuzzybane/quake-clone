A fully playable (maybe a bit buggy)  Multiplayer Arena Shooter made entirely using Gemini 3

If no one plays for a while the server will go to sleep as it is hosted on render's free tier

If it has gone to sleep just wait a minute and it will wake up and be fast again

The link to the game is
https://my-quake-clone.onrender.com/

the Server can handle 16 players 
Server Messages that states a player has joined, <playername> Fragged <playername> and <playername> has won

Share this repo to as many people as possible so we can all have some fun

please create issues if you encounter any bugs

WASD to move
Left Click to Fire
1,2,3 changes between Blaster, Shotgun and Railgun
